The "Pascal Complete" extension provides full support for Pascal development within Visual Studio Code. It includes advanced syntax highlighting, ready-made code snippets for creating common structures like programs, procedures, and loops, real-time error checking using the Free Pascal compiler, commands for quick compilation and execution of files, automatic code formatting, and integration with a Language Server Protocol (LSP) to improve autocompletion and code navigation. The extension supports .pas, .pp, .dpr files and more, enabling seamless work with environments like Delphi and Free Pascal.

## Installation

1. Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=PlayRood32.pascal).
2. Reload VS Code.
3. Open a Pascal file to activate.

## Requirements

- Free Pascal Compiler (FPC) for diagnostics and compilation.
